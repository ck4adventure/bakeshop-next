# Schedule

The **Schedule** screen is where you set your weekly production plan — how many batches of each item should be baked on each day of the week.

## How it works

Each row represents one item. Each column represents a day of the week. The number in each cell is the **default quota** for that item on that day.

- A blank cell means no baking is planned for that item on that day
- Quotas feed directly into the **Today** screen to show daily targets

## Editing a quota

1. Tap the cell for the item and day you want to change
2. Enter the new quantity
3. Changes save automatically

## Daily overrides

The schedule sets *default* weekly quotas. If you need to change the quota for a specific date without affecting the whole week, use the **override** feature on the Today screen instead.

## Tips

- Set quotas to reflect your normal production rhythm — you can always adjust on the fly from Today
- Zero out days your bakery is closed (or configure closed days in **Operating Days** to hide them automatically)
