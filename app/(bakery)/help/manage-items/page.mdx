# Manage Items

The **Manage Items** screen is available to **Admins and Managers**. Use it to create new items, edit existing ones, or remove items that are no longer in production.

## Item fields

| Field | Description |
|-------|-------------|
| **Name** | Display name shown throughout the app |
| **Category** | Groups items on the Inventory and Today screens |
| **Par level** | The target minimum stock — used to calculate status (low, critical) |
| **Default batch quantity** | How many units one bake produces — pre-fills the batch logging dialog |
| **Has inventory** | Uncheck for day-of items that don't need freezer tracking |

## Adding an item

1. Tap **Add Item** (top right)
2. Fill in the name and any optional fields
3. Save — the item appears immediately on Today and Inventory

## Editing an item

Tap any item row to open its edit sheet. Changes take effect immediately across the app.

## Deleting an item

Open an item's edit sheet and tap **Delete**. This removes the item from active views but its historical transactions remain in History.

## Categories

Categories are managed separately. If you need a new category, add it from the category field when creating or editing an item.
